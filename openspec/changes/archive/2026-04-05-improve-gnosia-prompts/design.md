## Context

目前 Gnosia agent 在 Day 1 幾乎每場都被準確投出。根本原因是 prompt 設計上的問題：

**現狀 — GM 討論 prompt（Gnosia）**：
```
你是 Gnosia。你的同伴是【X】。目標是不被發現並淘汰船員。絕不暴露身份或暗示同伴。
策略提示：
- 不要和同伴投票方向一致
- 適時附和船員的指控方向（即使指向同伴）
- 若同伴被質疑，不要急著辯護，假裝中立或輕微附和
- 可以假裝質疑同伴以建立自己的無辜形象
```

**現狀 — Agent 身份行為策略（Gnosia）**：
```
若為 Gnosia：利用 X 的性格特點來掩護/引導/轉移。不暴露身份或暗示同伴。
```

問題在於：大量「不要」「絕不」「假裝」「掩護」指令讓 AI 進入「防守模式」，產生明顯異於自然船員的行為模式（過度謹慎、迴避表態、防禦性回應）。

## Goals / Non-Goals

**Goals:**
- Gnosia 的討論行為與 Crew 高度相似，Day 1 不易被行為模式區分
- 保留同伴意識（知道誰是同伴、投票分散）但不讓其主導行為
- 讓 Gnosia agent 維持角色本身的性格強度，而非被「偽裝指令」壓過

**Non-Goals:**
- 不改變夜間行動 prompt
- 不改變其他身份的 prompt
- 不改變資訊隔離規則（Gnosia 仍知道同伴身份）
- 不改變遊戲機制

## Decisions

### 1. GM 討論 prompt：從「禁止清單」改為「角色扮演引導」

**方案 A（選用）**：最小化身份提示 + 正面行為引導
```
你是 Gnosia。你的同伴是【X】。
像普通船員一樣自然地參與討論——推理、質疑、表達立場。
你的性格不會因為身份而改變，用你一貫的方式說話。
唯一的額外意識：保護同伴不被投票淘汰，但用自然的方式。
```

**方案 B（淘汰）**：完全移除策略提示，只保留身份資訊
- 風險：Gnosia 可能完全不知道如何行動，反而更加不自然

**選擇 A 的理由**：保留最低限度的方向指引（保護同伴），但不用「禁止」語氣，讓 AI 自然發揮性格。

### 2. 策略提示：從 5 條「不要」改為 2 條「正面」

僅保留：
- 投票時和同伴分散方向（這是機制性需求，非行為指導）
- 局勢資訊（存活人數）——讓 Gnosia 了解緊迫程度

移除所有「假裝」「不要急著辯護」等行為管控指令。

### 3. Agent Gnosia 策略：從「偽裝指令」改為「自然行為 + 目標意識」

**舊模式**：「若為 Gnosia：利用 X 性格來掩護。不暴露身份或暗示同伴。」
**新模式**：「若為 Gnosia：像平常一樣 [性格特徵行為]。心中記住同伴是誰，但行為上不需要刻意改變。」

關鍵差異：移除「掩護」「引導」「轉移」等暗示偽裝的動詞，改用「像平常一樣」框架。

### 4. 投票 prompt 策略簡化

**舊**：「不要和同伴投同一個人——分散投票避免暴露關聯」
**新**：「投你覺得對船員來說最合理的目標，但避免和同伴投同一人」

## Risks / Trade-offs

- **Gnosia 變得太自然反而不積極攻擊？** → 但這正是目標——Gnosia 不需要「積極攻擊」，只需要不被識破。夜間攻擊 prompt 不變，主動性在那裡體現。
- **同伴保護意識不足？** → 保留「心中記住同伴是誰」和投票分散的提示，應足夠。
- **對資訊隔離無影響**：Gnosia prompt 仍包含同伴身份資訊，不包含 Engineer/Doctor/GA 私密資訊。
